--- layout: page title: Guide to libraries in propane --- It is possible to use vanilla [processing libraries][processing] with propane, but we also provide pure ruby libraries eg `boids`. Most libraries can be loaded using the [load_library][load_library] method, the exception being those that can be loaded as [gems][gems]. It is also possible use the [load_library][load_library] method to load local libraries for your sketches see [raining.rb][raining], if you stick with propane library conventions. Key To Library Types

  1. Builtin pure ruby eg boids
  2. Builtin wrapping java eg video_event
  3. Custom pure ruby
  4. Custom java (see also pbox2d gem)
  5. Regular processing libraries (see also gems for toxiclibs etc)
